## 🎯 PASOS FINALES PARA ACTIVAR

### Paso 1: Agregar Variables en Vercel (5 min)
```
Vercel Dashboard → Settings → Environment Variables
- NEXT_PUBLIC_SUPABASE_URL = Tu URL
- NEXT_PUBLIC_SUPABASE_ANON_KEY = Tu ANON KEY
```

### Paso 2: Crear Tablas en Supabase (2 min)
```
supabase.com → SQL Editor
Copiar contenido de /supabase/schema.sql
Ejecutar
```

### Paso 3: Deploy (1 min)
```
git push origin main
Esperar a que termine deployment en Vercel
```

### Paso 4: Test (5 min)
```
Abre @MARISOL ANCESTRAL BOT en Telegram
Verifica que funciona todo
```
